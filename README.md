# Property Nama CMS

This project is a Node.js + Express + Sequelize website with an admin panel for managing pages, blogs, and site settings.

## Run locally

1. Install dependencies: `npm install`
2. Start the app: `npm start`
3. Open http://localhost:3000/
4. Login to admin: username `admin`, password `admin123`

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repository in Vercel.
3. Set the build command to `npm install`.
4. Deploy.
